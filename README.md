# LongScope backend

LongScope backend services.

## Goals:

- download financial data from public APIs
- create plot data
- calculate financial markers
- expose an API

## Setup

This project uses poetry to manage dependencies.
First, install *Poeetry*:

```
pip install poetry
```

Install the project:

```
poetry install
```

Run:

```
poetry run python3 <FILE>
```
